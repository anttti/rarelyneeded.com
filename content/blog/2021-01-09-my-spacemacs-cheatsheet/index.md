## Configuration

- `SPC f e d`: Open .spacemacs
- `SPC f e R`: Reload .spacemacs

## treemacs

[Documentation](https://develop.spacemacs.org/layers/+filetree/treemacs/README.html)

- `SPC 0`: Open/focus file tree
- `SPC f t`: Toggle file tree
